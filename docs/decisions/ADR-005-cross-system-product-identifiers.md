# ADR-005: Cross-system product identifiers

**Status:** Proposed — Pascal approval required.

Adopt immutable lowercase kebab-case `product_code`, also used as Woo SKU: for example `eurofighter-typhoon-a2`. Titles, URLs, dimensions and prices can change; the code does not. Custom registry stores optional Woo product ID/URL. A product’s active selling mode determines price/stock/order authority, preventing duplicate editable commercial truth.
